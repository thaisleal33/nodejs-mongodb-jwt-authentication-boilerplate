
# Node.js + mongodb + jwt authentication

Boilerplate application for handling register, login, user deletion and update, password change and token authorization.


Made with:
- ❤ & ☕

Dependencies:
- bcrypt
- dotenv
- express
- jsonwebtoken
- mongoose

Dev dependencies:
- nodemon



## Install

```bash
  npm i
```


## Run

```bash
  npm run start
```
